# FRADyS — Formulario de Requisición de Material de Laboratorio

Versión 1.0

FRADyS es una aplicación frontend (React + Vite) cuyo propósito es facilitar la solicitud y gestión de material de laboratorio para docentes y estudiantes. Provee un formulario (Solicitud) con campos para seleccionar materiales, fechas y cantidades, e incluye una interfaz limpia y responsiva para su uso en escritorio y dispositivos móviles.

Características principales
- Formulario de solicitud de material de laboratorio.
- Interfaz responsiva y accesible.
- Componentes reutilizables para listas de materiales y selección de horarios.

Stack
- React
- Vite
- Tailwind CSS (opcional según configuración del proyecto)

Instalación y ejecución (desarrollo)
1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev
```

Estructura relevante
- `src/Pages/SolicitudPage.jsx` — página principal del formulario.
- `src/Components/MaterialForm/` — componentes relacionados al formulario.
- `src/assets/` — imágenes y recursos locales (pueden estar ignorados en .gitignore si se desea).
